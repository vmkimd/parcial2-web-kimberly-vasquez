## Descripción del Proyecto
Este es un proyecto Full Stack desarrollado para el segundo parcial. Incluye una arquitectura cliente-servidor con las siguientes tecnologías:
- **Backend:** Node.js con Express y PostgreSQL.
- **Frontend:** Angular (Arquitectura Standalone).

## Requisitos Previos
1. Tener instalado **Node.js**.
2. Tener una instancia de **PostgreSQL** funcionando.
3. Crear las tablas necesarias (`usuarios` y `perfiles`) con el script SQL adjunto en el documento del parcial.

## Instrucciones para Ejecución

### Configuración del Backend
1. Navegar a la carpeta `backend`: `cd backend`
2. Instalar dependencias: `npm install`
3. Iniciar el servidor: `node server.js`
   - El servidor correrá en `http://localhost:3000`.

### Configuración del Frontend
1. Navegar a la carpeta `frontend`: `cd frontend`
2. Instalar dependencias: `npm install`
3. Iniciar la aplicación: `ng serve`
   - Acceder desde el navegador a `http://localhost:4200`.
